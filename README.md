# md2csv

Convert markdown table to csv file

## Install
```sh
npm install md2csv -g
```

## Usage

The test.md can be converted to test.csv using the either of the following ways.

### Command Line
```sh
md2csv test.md
```

### Javascrip

```js
const md2csv = require('md2csv');
md2csv('./test.md');
```
